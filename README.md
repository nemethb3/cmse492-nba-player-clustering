# Clustering NBA Players by Playing Style

This project aims to cluster NBA players based on their playing style using unsupervised machine learning techniques. 

## About nba_api

The `nba_api` is a Python package that allows users to access a wealth of NBA statistics and data. This package is useful for analyzing player performances and other related metrics. 

## Installation Instructions

To install the required packages, you can use pip:
```
pip install nba_api
```

## Project Structure

- `data/` - Contains datasets used for analysis
- `notebooks/` - Jupyter notebooks for exploratory data analysis and model building
- `src/` - Source code implementing the clustering algorithms

## Requirements

- Python >= 3.6
- nba_api
- numpy
- pandas
- scikit-learn
